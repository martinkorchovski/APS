## Задача 1
Дадено ви е едно нанижано бинарно дрво. Потребно е да пронајдете колкав е бројот на
дедо-внук релации во тоа дрво. 

Дедо-внук релација постои помеѓу два јазли A и B кај
кои што едно од децата на A е родител на B. Не смеете да користите рекурзија.

```
Input
7
0 1 ROOT
1 2 LEFT 0
2 3 RIGHT 0
3 4 LEFT 1
4 5 RIGHT 1
5 6 LEFT 2
6 7 RIGHT 2
Result
4
```
