## Задача 5
Дадена е еднострано поврзана листа. Да се напише функција која од листата ќе ги избрише сите елементи кои во data полето имаат непарен број. Како резултат функцијата да враќа колку јазли биле избришани, како и да ја печати листата.
```
Input 
8 
4 3 7 6 4 7 3 2 
Result 
4->6->4->2 
4
```
