## Задача 8
Дадена е еднострано поврзана листа. Да се напише функција која ќе ги избрише сите повеќекратни елементи од листата (при секоја појава на два соседни елементи со иста информација едниот треба да се избрише). Како резултат функцијата да враќа колку јазли биле избришани, како и да ја печати листата.
```
Input 
11 
3 4 4 7 6 4 4 4 7 3 2 
Result 
3->4->7->6->4->7->3->2 
3
```
