## Задача 3
Дадена е еднострано поврзана листа. Да се напише функција која во листата ќе го пронајде последниот јазол со информација x и ќе го префрли на крајот на листата. Како резултат функцијата да ја враќа позицијата (броено од почетокот на листата) на која бил пронајден бараниот јазол или -1 ако таков јазол не бил пронајден во листата.
```
Input 
8 
3 4 7 6 4 7 3 2 7 
Result 
3->4->7->6->4->3->2->7 
6
```
