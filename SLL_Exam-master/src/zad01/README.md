## Задача 1
 Дадена е еднострано поврзана листа. Да се напише функција која во листата ќе го 
 пронајде првиот јазол со информација x и ќе го префрли на почетокот на листата. 
 Како резултат функцијата ја враќа позицијата (броено од почетокот на листата) на која бил 
 пронајден бараниот јазол или -1 ако таков јазол не бил пронајден во листата.
```
Input 
8
3 4 7 6 4 7 3 2
7
Result
7->3->4->6->4->7->3->2
3
 ```
