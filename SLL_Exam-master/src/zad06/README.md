## Задача 6
Дадена е еднострано поврзана листа. Да се напише функција која пред секој елемент кој во data полето има вредност x ќе вметне нов јазел со информација y. Како резултат функцијата да враќа колку јазли биле избришани, како и да ја печати листата.
```
Input 
8 
4 3 7 6 4 7 3 2 
3 
1 
Result 
4->1->3->7->6->4->7->1->3->2 
2
```
