## Задача 1
Дадена е двојно поврзана листа од двојно поврзани листи. Да се најде сума на секоја од подлистите, а потоа производ на овие суми.

Влез: Број N кој означува колку листи има. Број M кој означува колку елементи има во секоја листа. Во следните М линии се податоците 1<=A<=1000 за секоја од листите.

Излез: Еден број што е производот од сумите од низите. Со седум децимали.
```
Input 
3 
4 
1 2 3 4 
2 3 4 5 
6 7 8 9 
Result 
4200
```
